# MomentKeep 后端开发文档

## 项目概述
MomentKeep（朝暮记）后端服务基于Spring Boot 3.5.13开发，使用TiDB Cloud作为数据库，提供完整的用户管理、打卡、待办、倒计时等功能，并集成DeepSeek AI服务。

## 技术栈
- **后端框架**: Spring Boot 3.5.13
- **Java版本**: JDK 21
- **数据库**: TiDB Cloud (MySQL兼容)
- **ORM框架**: MyBatis-Plus
- **缓存**: Redis
- **对象存储**: 缤纷云存储 (S3兼容)
- **AI服务**: DeepSeek API
- **安全框架**: Spring Security + JWT
- **构建工具**: Maven
- **日志框架**: SLF4J + Logback

## 核心功能模块

### 1. 用户管理模块 (User)

#### 实体类 (User.java)
```java
@Data
@TableName("user")
public class User {
    @TableId(type = IdType.AUTO)
    private Long id;
    private String username;       // 用户名（唯一标识）
    private String nickname;        // 昵称
    private String email;          // 邮箱
    private String phone;          // 手机号
    private String avatar;         // 头像URL
    private String password;       // 密码（加密存储）
    private LocalDateTime createTime;  // 创建时间
    private LocalDateTime updateTime;  // 更新时间
    private Integer status;        // 状态（1正常，-1已注销）
    private String role;           // 角色（USER/ADMIN）
    @Version
    private Integer version;       // 乐观锁版本号
}
```

#### 主要接口
| 接口 | 方法 | 路径 | 描述 |
|------|------|------|------|
| register | POST | /api/user/register | 用户注册 |
| login | POST | /api/user/login | 用户登录 |
| getProfile | GET | /api/user/profile | 获取用户信息 |
| updateProfile | PUT | /api/user/profile | 更新用户信息 |
| uploadAvatar | POST | /api/user/avatar | 上传头像 |
| logout | POST | /api/user/logout | 用户登出 |
| delete | POST | /api/user/delete | 注销账户 |
| changePassword | POST | /api/user/change-password | 修改密码 |

## 安全机制

### JWT认证流程
1. 用户登录成功后，服务器生成JWT Token返回给客户端
2. 客户端在后续请求的Header中携带Token
3. 服务器通过JwtAuthenticationFilter验证Token的有效性
4. 验证通过后，将用户信息存入SecurityContextHolder

### 安全配置要点
- 登录接口（/api/user/login）和注册接口（/api/user/register）无需认证
- 管理后台接口（/api/admin/**）需要ADMIN角色
- 其他接口需要有效Token
